## ⚠️ 常见问题快速解答

老师问什么系统就发什么：

### Q: "我不知道我用的什么系统"
**A:**
```
老师您好！简单判断方法：
- 如果电脑桌面左下角有"开始"按钮 → 用 Windows版
- 如果屏幕底部中间有一排彩色图标 → 用 Mac版

或者直接两个版本都给您发过来，您两个都试试，能用的就是对的版本😊
```

### Q: "双击后没反应"
**Windows:**
```
右键"智能启动.bat" → 以管理员身份运行
```

**Mac:**
```
右键文件 → 打开 → 确认打开
（首次使用需要授权，之后就正常了）
```

### Q: "提示未安装Python"（仅Windows）
```
访问这个网址下载安装：
https://www.python.org/downloads/

安装时记得勾选 "Add Python to PATH"
安装完成后重新双击启动文件就可以了
```

### Q: "数据会丢失吗"
```
不会的！数据保存在您的电脑浏览器中
只要用同一台电脑和浏览器打开，数据一直都在

建议您定期复制整个文件夹备份
也可以复制到U盘在其他电脑使用
```
