# 睿析成绩分析系统 - 教师使用指南

## 📖 快速开始（3步上手）

### 第一步：打开软件
- **Windows用户**：双击桌面上的"睿析成绩分析系统"图标
- **Mac用户**：在启动台中找到"睿析成绩分析系统"并打开

### 第二步：下载模板
1. 点击"配置科目并下载模板"按钮
2. 根据您的需要添加或删除科目（语文、数学、英语必须保留）
3. 设置每个科目的满分（默认100分）
4. 点击"确认并下载"按钮

### 第三步：填写成绩并导入
1. 打开下载的Excel模板
2. 按照示例格式填写学生成绩
3. 保存Excel文件
4. 在软件中点击"上传成绩表"
5. 输入考试名称（如"期中考试"）
6. 点击"确认导入"

🎉 **恭喜！** 软件会自动生成成绩分析报告，您可以：
- 查看整体成绩分析（统计、图表、排名）
- 查看单个学生的详细报告
- 对比多次考试的成绩变化

## 📋 详细功能说明

### 1️⃣ 导入成绩
**位置**：点击左侧菜单的"导入成绩表"

**步骤**：
1. 先下载模板
2. 在Excel中填写成绩
3. 上传填好的Excel文件

**注意事项**：
- ✅ 必须包含"姓名"列
- ✅ 可选包含"学号"和"班级"列
- ✅ 缺考的学生请留空，不要填0
- ❌ 不要修改模板的表头（第一行）

### 2️⃣ 成绩分析
**位置**：点击左侧菜单的"成绩分析"

**包含内容**：
- 📊 整体统计数据（平均分、及格率、优秀率）
- 📈 各科成绩分布图表
- 🏆 班级排名（按总分）
- 📉 各科通过情况分析

### 3️⃣ 学生报告
**位置**：点击左侧菜单的"学生报告"

**功能**：
- 🔍 搜索学生姓名
- 📄 查看单个学生的详细成绩单
- 💾 导出学生报告为PDF（可打印）

### 4️⃣ 历史记录
**位置**：点击左侧菜单的"历史记录"

**功能**：
- 📚 查看所有导入过的考试
- 🔄 对比两次考试的成绩变化
- 📈 查看学生的成绩趋势（需要导入多次考试）
- 🗑️ 删除不需要的历史记录

**对比两次考试**：
1. 点击"考试对比"按钮
2. 从下拉框中选择"考试A"
3. 从下拉框中选择"考试B"
4. 系统自动显示对比结果

## 💡 常见问题解答

### Q1：软件需要联网吗？
**A**：不需要！软件完全离线使用，所有数据都保存在您的电脑上。

### Q2：我的数据安全吗？
**A**：非常安全！所有成绩数据都只保存在您的电脑上，不会上传到任何服务器。

### Q3：如何修改已经导入的成绩？
**A**：目前不支持直接修改，请重新导入。建议先删除旧记录，再导入新的成绩表。

### Q4：一次可以导入多少学生的成绩？
**A**：理论上没有限制，测试过500名学生以内都能流畅运行。

### Q5：成绩趋势为什么看不到？
**A**：需要满足以下条件：
1. 至少导入2次考试的数据
2. 学生姓名必须完全一致（包括空格和标点符号）
3. 在历史记录页面搜索学生姓名

### Q6：如何卸载软件？
**Windows**：控制面板 → 卸载程序 → 选择"睿析成绩分析系统" → 卸载
**Mac**：将应用拖入废纸篓

### Q7：软件突然打不开了怎么办？
**A**：请尝试：
1. 重启电脑后再打开
2. 检查是否被杀毒软件拦截
3. 重新安装软件
