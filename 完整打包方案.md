**适合人群**：所有老师（强烈推荐）

**交付内容**：

## 🚀 方案A：Electron桌面应用打包步骤

### 第一步：准备打包环境

```bash
cd /Users/lijiahao/Desktop/project/grade-analysis-system

# 1. 安装依赖（如果还没安装）
npm install

# 2. 构建Web应用
npm run build

# 3. 安装Electron打包工具（如果还没安装）
npm install --save-dev electron electron-builder

# 4. 验证构建结果
ls -lh dist/index.html  # 应该看到约2MB的文件
```

### 第二步：打包应用

**打包Windows版本：**
```bash
npm run package:win
```

**打包Mac版本：**
```bash
npm run package:mac
```

**同时打包两个版本：**
```bash
npm run package:all
```

### 第三步：测试安装包

**Windows测试：**
1. 找到 `release/睿析成绩分析系统_1.0.0_Windows.exe`
2. 双击安装
3. 测试所有功能
4. 确认数据持久化正常

**Mac测试：**
1. 找到 `release/睿析成绩分析系统_1.0.0_Mac.dmg`
2. 双击打开，拖拽到Applications
3. 测试所有功能

### 第四步：准备交付文件

```bash
# 创建交付文件夹
mkdir -p ~/Desktop/睿析成绩分析系统_交付包

# 复制安装包

# （使用Typora或其他Markdown转PDF工具）

cd ~/Desktop
zip -r 睿析成绩分析系统_桌面版.zip 睿析成绩分析系统_交付包/
```

## ⚡ 快速打包命令（一键完成）

创建以下脚本文件 `打包脚本.sh`:

```bash
#!/bin/bash

echo "======================================"
echo "  睿析成绩分析系统 - 自动打包脚本"
echo "======================================"
echo ""

# 1. 构建Web应用
echo "📦 正在构建Web应用..."
npm run build

# 2. 准备网页版
echo "📱 正在准备网页版..."
mkdir -p ~/Desktop/睿析成绩分析系统_网页版
cp dist/index.html ~/Desktop/睿析成绩分析系统_网页版/睿析成绩分析系统.html
cp 启动程序.bat ~/Desktop/睿析成绩分析系统_网页版/
cp 启动程序.command ~/Desktop/睿析成绩分析系统_网页版/

# 3. 打包桌面版（如果需要）
read -p "是否打包桌面版？(y/n): " choice
if [ "$choice" = "y" ]; then
    echo "🖥️  正在打包桌面版..."
    npm run package:all
fi

# 4. 创建压缩包
echo "📦 正在创建压缩包..."
cd ~/Desktop
zip -r 睿析成绩分析系统_网页版_$(date +%Y%m%d).zip 睿析成绩分析系统_网页版/

echo ""
echo "✅ 打包完成！"
echo "📁 文件位置：~/Desktop/睿析成绩分析系统_网页版_$(date +%Y%m%d).zip"
echo ""
```

使用方法：
```bash
chmod +x 打包脚本.sh
./打包脚本.sh
```
